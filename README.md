# Homebrew Tap

Personal Homebrew tap for my apps.

## Installation

```bash
brew tap shockz09/tap
brew install --cask typingstats
```

## Available Casks

| Cask | Description |
|------|-------------|
| `tapped` | Minimal macOS menubar app that tracks daily keystrokes with iCloud sync |
| `typingstats` | Alias for tapped |
